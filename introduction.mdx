## What is ZeroDash?

ZeroDash is a **lightweight, fully static homelab dashboard** designed for fast navigation to services and applications, with optional visual monitoring — all **without running a backend, database, or plugins**.

It's a static replacement for dashboards like Homarr and Dashy, built with a different philosophy: keep the server simple, push intelligence to the browser, and remain easy to deploy and maintain long-term.

## Key features

### Grid-based layouts

Define precise tile positions using a CSS Grid system. Each layout specifies rows and columns, and tiles declare their position and size.

```javascript
{
  name: "desktop",
  rows: 4,
  columns: 6,
  tiles: [
    {
      type: "application",
      layout: { x: 0, y: 0, width: 2, height: 2 },
      config: { label: "Jellyfin", url: "https://jellyfin.local" }
    }
  ]
}
```

### Global search

Press any key to open the command palette. Search across all tiles with prefix shortcuts:
- `@` for applications
- `#` for applications and links  
- `/` for buttons
- `.` for toggles

### Availability monitoring

Application and link tiles can monitor service availability with automatic status checks.

## Why ZeroDash?

### Simple deployment

No Docker containers, no databases, no background processes. Just HTML, CSS, and JavaScript files that you can deploy anywhere.

### Long-term maintainability

Built with web standards that won't change. No dependency updates, no breaking changes, no version conflicts.

### Browser-first architecture

All intelligence runs in the browser:
- Status checks use the Fetch API
- Search indexing happens on page load
- Layout selection uses media queries
- Text fitting uses DOM measurements

### Configuration as code

Your dashboard is defined in a single JavaScript configuration file that exports a simple object structure. Version control it, back it up, share it.
